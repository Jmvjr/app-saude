Solicitação de Exclusão de Dados Pessoais

Aplicativo: SAÚDE!
Desenvolvedor: br.unicamp.ic.saude

Esta página destina-se aos usuários que desejam exercer seu direito à exclusão de dados pessoais, conforme previsto na Lei Geral de Proteção de Dados Pessoais (LGPD) e demais regulamentações aplicáveis.

Procedimento para Exclusão Total da Conta

Método 1: Exclusão via Aplicativo

  1. Acesse o aplicativo SAÚDE!
  2. Navegue até a seção "Perfil"
  3. Selecione a opção "Deletar Conta"
  4. Confirme a operação conforme solicitado

Prazo de Processamento: A exclusão será processada imediatamente, resultando na remoção permanente e irreversível de todos os dados associados à conta.

Solicitação de Exclusão Parcial de Dados

Caso deseje manter sua conta ativa mas solicitar a exclusão de categorias específicas de dados, favor encaminhar solicitação formal através do canal oficial de atendimento.

Canal de Atendimento: saude.app.unicamp@gmail.com

Assunto da Mensagem: Solicitação de Exclusão Parcial de Dados Pessoais

Informações Obrigatórias:

  • Endereço de e-mail registrado na plataforma
  • Identificação específica das categorias de dados a serem excluídas
  • Justificativa para a solicitação (opcional)

Prazo de Atendimento: Até 30 (trinta) dias úteis a partir do recebimento da solicitação.

Categorias de Dados Sujeitas à Exclusão

As seguintes categorias de dados pessoais serão removidas mediante solicitação de exclusão:

  • Dados de identificação e cadastro (nome completo, endereço de e-mail)
  • Informações de perfil do usuário
  • Registros do diário pessoal
  • Dados de saúde e bem-estar registrados
  • Histórico de interações e preferências
  • Informações compartilhadas com profissionais de saúde credenciados

Dados Mantidos Após a Exclusão

Conforme previsto na legislação aplicável, serão mantidos exclusivamente:

  • Dados estatísticos agregados e anonimizados para fins de pesquisa e desenvolvimento
  • Informações necessárias para cumprimento de obrigações legais
  • Registros de auditoria e segurança, quando aplicável

Importante: Os dados mantidos não permitirão a identificação individual do usuário.

Prazos de Retenção

  • Exclusão via aplicativo: Processamento imediato
  • Solicitação por e-mail: Até 30 (trinta) dias úteis

Informações Complementares

Para consultar informações detalhadas sobre o tratamento de dados pessoais, recomendamos a leitura de nossa Política de Privacidade.

Em caso de dúvidas adicionais ou necessidade de esclarecimentos sobre este procedimento, entre em contato através dos canais oficiais de atendimento.
